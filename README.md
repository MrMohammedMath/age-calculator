### **Usage**

```bash
  git clone https://github.com/MrMohammedMath/age-calculator.git

  cd age-calculator

  pnpm install
```

- for running dev server

```bash
pnpm run dev
```

- for build & review

```bash
pnpm run build
pnpm run preview
```
